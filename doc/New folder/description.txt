Here is the basic overview of the project.

Efficient vehicle assistance is essential for ensuring road safety, enhancing convenience, and minimizing downtime, particularly in Sri Lanka, where many small garages operate without real-time coordination. Specially , ladies and under 25 ages don't know about the vehicle faults even they don't have knowledge of the technical part of the car. This project presents an AI-driven mobile application that uses a conversational chatbot to guide vehicle owners in reporting car issues by natural language and uploading photos of the dashboard, determining whether the vehicle is drivable, and arranging repairs. Users describe problems such as â€œMy engine is overheatingâ€ or â€œBrake not working,â€ and by adding images of dashboard and the chatbot, built with Gemini API, identifies the fault category and type. It then asks if the car can be driven, branching into two paths: if drivable to garage, the system queries nearby garages via Google Maps API for travel distance and ETA, then applies a ranking algorithm to suggest the best garages based on availability (available or busy or closed), mechanics and their experience, user ratings, distance and arrival time; if not drivable to garage, the platform dispatches the nearest available mechanic with real-time ETA and location tracking. Users can book services, track repair progress, and provide feedback. By combining conversational AI, predictive modeling, and location-based services, this system overcomes limitations in existing Sri Lankan solutions that offer static garage listings without intelligent recommendations or dispatch support. The application enhances convenience, reduces waiting times, and fosters trust among vehicle owners, mechanics, and garages. Furthermore, the data collected enables workshops to optimize operations and improve service efficiency. Overall, this project presents the inaugural all-encompassing intelligent garage recommendation and repair time estimation platform in Sri Lanka, thereby enhancing the accessibility, efficiency, and reliability of vehicle assistance.

### 4.4 Module 4: Garage Ranking & Recommendation Engine

**Complete Ranking Algorithm:**
```javascript
class GarageRankingEngine {
    constructor() {
        this.weights = {
            distance: 0.4,        // 40% - Proximity is critical
            waiting_time: 0.3,    // 30% - User wants fast service
            rating: 0.2,          // 20% - Quality matters
            availability: 0.1     // 10% - Mechanic count
        };
    }
    
    async rankGarages(userLocation, diagnosis, radius_km = 10) {
        // Step 1: Get nearby garages from database
        const nearbyGarages = await this.getNearbyGarages(
            userLocation.lat,
            userLocation.lng,
            radius_km
        );
        
        // Step 2: Filter by specialization
        const specializedGarages = nearbyGarages.filter(garage => 
            garage.specialties.includes(diagnosis.fault_category)
        );
        
        // Step 3: Calculate scores for each garage
        const scoredGarages = await Promise.all(
            specializedGarages.map(garage => this.calculateScore(garage, userLocation, diagnosis))
        );
        
        // Step 4: Sort by composite score (lower is better)
        scoredGarages.sort((a, b) => a.composite_score - b.composite_score);
        
        // Step 5: Return top 5
        return scoredGarages.slice(0, 5);
    }
    
    async calculateScore(garage, userLocation, diagnosis) {
        // Calculate distance using Haversine formula
        const distance_km = this.calculateDistance(
            userLocation.lat, userLocation.lng,
            garage.latitude, garage.longitude
        );
        
        // Get drive time from Google Maps API
        const driveTimeMinutes = await this.getDriveTime(
            userLocation, 
            {lat: garage.latitude, lng: garage.longitude}
        );
        
        // Get current queue time from garage portal
        const queueMinutes = garage.is_available ? garage.current_queue_minutes : 0;
        
        // Calculate component scores
        const distanceScore = distance_km;
        const waitScore = (queueMinutes + driveTimeMinutes) / 60;
        const ratingScore = 5 - garage.rating;
        const availabilityScore = 5 - garage.available_mechanics;
        
        // Calculate composite score
        const compositeScore = (
            (distanceScore * this.weights.distance) +
            (waitScore * this.weights.waiting_time) +
            (ratingScore * this.weights.rating) +
            (availabilityScore * this.weights.availability)
        );
        
        return {
            garage_id: garage.id,
            name: garage.name,
            address: garage.address,
            phone: garage.phone,
            rating: garage.rating,
            total_reviews: garage.total_reviews,
            distance_km: parseFloat(distance_km.toFixed(1)),
            drive_time_minutes: driveTimeMinutes,
            queue_wait_minutes: queueMinutes,
            total_wait_minutes: queueMinutes + driveTimeMinutes,
            available_mechanics: garage.available_mechanics,
            is_available: garage.is_available,
            specialties: garage.specialties,
            
            // Score components
            distance_score: distanceScore,
            wait_score: waitScore,
            rating_score: ratingScore,
            availability_score: availabilityScore,
            composite_score: parseFloat(compositeScore.toFixed(3)),
            
            // Estimated cost (from ML prediction + garage markup)
            estimated_cost: this.estimateCost(diagnosis, garage.garage_type),
            
            // Google Maps data
            maps_url: `https://www.google.com/maps/dir/?api=1&destination=${garage.latitude},${garage.longitude}`,
            
            // Rank will be assigned after sorting
            rank: null
        };
    }
    
    calculateDistance(lat1, lon1, lat2, lon2) {
        // Haversine formula for accurate distance
        const R = 6371; // Earth radius in km
        const dLat = (lat2 - lat1) * Math.PI / 180;
        const dLon = (lon2 - lon1) * Math.PI / 180;
        const a = 
            Math.sin(dLat/2) * Math.sin(dLat/2) +
            Math.cos(lat1 * Math.PI / 180) * Math.cos(lat2 * Math.PI / 180) *
            Math.sin(dLon/2) * Math.sin(dLon/2);
        const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
        return R * c;
    }
    
    async getDriveTime(origin, destination) {
        // Google Maps Distance Matrix API
        const response = await googleMaps.distanceMatrix({
            origins: [`${origin.lat},${origin.lng}`],
            destinations: [`${destination.lat},${destination.lng}`],
            mode: 'driving',
            traffic_model: 'best_guess',
            departure_time: 'now'
        });
        
        return response.rows[0].elements[0].duration.value / 60; // Convert to minutes
    }
    
    estimateCost(diagnosis, garageType) {
        // Base cost from parts + labor database
        const baseCost = this.getBaseCost(diagnosis.fault_type);
        
        // Markup for authorized garages
        const markup = garageType === 'Authorized' ? 1.2 : 1.0;
        
        return Math.round(baseCost * markup);
    }
}
```

```javascript
// Server-side (Node.js + Socket.io)
const io = require('socket.io')(server, {
    cors: { origin: "*" }

// Namespace for job tracking
const jobTracking = io.of('/tracking');

jobTracking.on('connection', (socket) => {
    console.log(`Client connected: ${socket.id}`);
    
    // Driver subscribes to job updates
    socket.on('subscribe_job', async ({ job_id, user_id }) => {
        // Verify user owns this job
        const job = await Job.findOne({ _id: job_id, user_id });
        if (!job) {
            socket.emit('error', { message: 'Unauthorized' });
            return;
        }
        
        // Join room for this job
        socket.join(`job_${job_id}`);
        
        // Send current status
        socket.emit('current_status', {
            job_id,
            status: job.status,
            updates: job.updates,
            last_update: job.updated_at
        });
    });
    
    // Garage/Mechanic updates job status
    socket.on('update_status', async ({ job_id, status, note, photos }) => {
        const job = await Job.findById(job_id);
        
        // Add update to job history
        job.updates.push({
            status,
            note,
            photos: photos || [],
            timestamp: new Date(),
            updated_by: socket.user_id
        });
        
        job.status = status;
        job.updated_at = new Date();
        await job.save();
        
        // Broadcast to all subscribers (driver + admin)
        jobTracking.to(`job_${job_id}`).emit('status_changed', {
            job_id,
            status,
            note,
            photos,
            timestamp: new Date()
        });
        
        // Send push notification to driver
        await sendPushNotification(job.user_id, {
            title: `Job ${job.booking_number} Updated`,
            body: note || `Status changed to: ${status}`,
            data: { job_id, status }
        });
    });
    
    // Live location tracking (mechanic only)
    socket.on('update_location', async ({ job_id, latitude, longitude, heading }) => {
        // Verify user is assigned mechanic
        const job = await Job.findOne({ _id: job_id, mechanic_id: socket.user_id });
        if (!job) return;
        
        // Broadcast location to driver
        jobTracking.to(`job_${job_id}`).emit('mechanic_location', {
            latitude,
            longitude,
            heading,
            timestamp: new Date()
        });
        
        // Calculate new ETA
        const driverLocation = await getDriverLocation(job.user_id);
        const eta = await calculateETA(
            { latitude, longitude },
            driverLocation
        );
        
        // Update ETA
        jobTracking.to(`job_${job_id}`).emit('eta_updated', {
            eta_minutes: eta,
            distance_km: await calculateDistance(
                { latitude, longitude },
                driverLocation
            )
        });
    });
    
    socket.on('disconnect', () => {
        console.log(`Client disconnected: ${socket.id}`);
    });
});
```
